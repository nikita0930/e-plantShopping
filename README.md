# 🌿 e-plantShopping

## 📌 Project Overview

e-plantShopping is a React-based web application that allows users to browse plants and manage a shopping cart. It uses **Redux Toolkit** for global state management and provides a smooth shopping experience with real-time updates.

---

## 🚀 Features

* 🛒 Add plants to cart
* 🔄 Increase / decrease quantity
* ❌ Remove items from cart
* 💰 Automatic total price calculation
* 🔢 Live cart item count
* 🔒 Disable "Add to Cart" after adding
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

* React.js
* Redux Toolkit
* JavaScript (ES6+)
* HTML5 & CSS3
* Vite

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/nikita0930/e-plantShopping.git
```

2. Navigate to project folder:

```
cd e-plantShopping
```

3. Install dependencies:

```
npm install
```

4. Run the app:

```
npm run dev
```

---

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

To deploy:

```
npm run deploy
```
